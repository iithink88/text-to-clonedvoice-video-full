🎬 文案秒变短视频

你只要写一段"口播文案"（就像发朋友圈的长文字），剩下的配音、字幕、画面、剪辑这个工具全帮你搞定，最后吐出一条 mp4 短视频。本指南专为"不想看说明书"的朋友设计，照着做 3 步就能出片。

朋友分享版 · 免费声零配置 / 克隆声约 10 元

准备 一次性装好环境（约 15 分钟，只做一次）

下面 4 件东西装好之后，以后每次做视频只要 3 步、点几下鼠标就行。

① Node.js
去 nodejs.org 下载 LTS 版，一路"下一步"安装
② ffmpeg
按 SKILL.md 里的链接下载，把 exe 放进系统 PATH
③ Python 依赖
命令行进本文件夹，跑 pip install -r requirements.txt
④ Vosk 模型
下载中文模型小包，解压到 models/vosk-model-small-cn-0.22
💡 不确定装没装好？双击 启动.bat，它会告诉你差哪一步。详情都在同包的 SKILL.md 里。

3 步 照着做，出片！

环境装好后，每次做视频就是下面这 3 步。

1

写文案

打开 input/文案.txt，把你的口播词写进去。想让某句话变橙色重点，用 **关键词** 包起来就行。不会写也行 —— 文件夹里已带一份示例文案，双击 示例一键出片.bat 就能直接出片看效果。

2

双击启动

直接双击本文件夹里的 启动.bat。会弹出一个图形窗口（不是黑框）：选声音引擎（A 克隆 / B 免费声）、挑音色、填加速倍数、可填标题，点「开始生成」即可。窗口下方有实时日志，做完弹窗告知成片位置。嫌麻烦？直接双击 示例一键出片.bat，一句话都不用输。

3

选好参数，等出片

接着选加速倍数（默认 1.3，让语速更利落）、填个标题，点「开始生成」。工具会自动生成配音 → 对齐字幕 → 渲染画面，几分钟後成片出现在你的桌面。

问答 常见问题

Q：双击启动.bat 没反应 / 一闪就关了？
A：两个常见原因——① 文件夹是从 zip 解压的，Windows 可能拦截脚本。右键 启动.bat → 属性 → 底部「解除锁定」(Unblock) 打勾 → 确定，再双击即可。② Python / ffmpeg 没装好，窗口里的日志会写清楚缺什么。仍不行就把窗口截图发我。
Q：想用自己的声音（克隆）怎么弄？
A：选 [A]，按 SKILL.md 里的"新手安装指导"配一下 MiniMax（首次克隆约 10 元，之后免费）。
Q：字幕对不上朗读节奏？
A：不会。字幕是用本地 Vosk 从真实音频识别出来的，天然对齐，不会飘。
Q：Mac / Linux 能用吗？
A：本分享包主要适配 Windows。其它系统可参考 SKILL.md 手动跑 scripts。
