# 文案秒变短视频 · text-to-clonedvoice-video-full

把一段口播文案，自动变成一条带 **配音 + 字幕 + 分镜画面** 的短视频（1920×1080，深色科技风）。

## ✨ 特性
- **双声音引擎，任选其一**：
  - **[B] edge-tts 免费声（默认）**：微软在线 TTS，零配置、零费用，立即可用。
  - **[A] MiniMax 克隆声**：真实克隆你的声音，首次约 10 元，之后永久免费生成。
- 字幕用本地 **Vosk**（Kaldi）从真实音频识别，天然对齐朗读节奏、不飘。
- 一条命令出片，适合做产品介绍、知识口播、课程短视频。

## 🚀 快速开始（三步走）
1. **装环境（一次）**：Node.js（LTS）、ffmpeg、Python 依赖 `pip install -r requirements.txt`、下载 Vosk 中文模型。
2. **写文案**：放进 `input/文案.txt`（用 `**重点**` 包起来可标橙色）。
3. **图形界面出片**：双击 `启动.bat` → 弹出图形窗口，选声音引擎（A 克隆 / B 免费声）、音色、加速倍数、标题 → 点「开始生成」→ 成片输出桌面。
   - 想一句话都不输？双击 **`示例一键出片.bat`**，用自带示例文案直接出片。

更详细的技术说明见 [`SKILL.md`](SKILL.md) 与 [`新手安装指导.md`](新手安装指导.md)。
